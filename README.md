# Mi(cro) Blog App

**Mi Blog App** es una plantilla para blog personal simple de usar, ligera, rápida y libre que está creado en El Reino de España.

**Mi Blog App** se desarrolla con las siguientes fantasticas tecnologías:

* Ruby
* Sinatra
* SQLite
* Bootstrap
* jQuery
